# fyp-frontend
###### This is food delivery app with alot of outofbox functionality
**Key Features**<br/>
- Google authentication
- Google maps integration
- Email verification
- Live Chat
- Datascience moderation model integration while uploading image
## To run 
> yarn install<br/>
> expo start --android/ios/web<br/>
